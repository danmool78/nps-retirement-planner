# 부부 노후자금 통합 시뮬레이터

국민연금·주택연금·추납·임의가입·물가상승률·기대수명을 통합하여
부부의 은퇴 후 현금흐름을 **월 단위로 시뮬레이션**하고, 여러 전략 조합 중
최적안을 찾아 시각화하는 Streamlit 웹앱입니다.

## 주요 기능
- 국민연금 조기/연기수령, 추납, 임의가입 반영
- 주택연금 개시나이별 월지급액 계산
- 물가·투자수익·기대수명을 반영한 월 단위 현금흐름 시뮬레이션
- **3가지 성향별 추천**: 🛡️안정형 · 📈총수령액 극대화형 · 🎁상속중시형
- 그래프 8종(현금흐름, 누적자산, 수령시점 비교, 히트맵, Pareto Frontier 등)
- CSV / Excel 내보내기

## 로컬 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 파일 구성
| 파일 | 역할 |
|---|---|
| `app.py` | Streamlit UI |
| `config.py` | 제도 파라미터·사용자 입력 정의(화면에서 수정 가능) |
| `nps.py` | 국민연금 계산 |
| `housing_pension.py` | 주택연금 계산 |
| `cashflow.py` | 월 단위 현금흐름 시뮬레이션 |
| `optimizer.py` | 조합 전수평가·점수화·파레토·민감도 |
| `visualization.py` | plotly 그래프 |
| `export.py` | CSV/Excel 내보내기 |

## 주의
본 도구의 국민연금·주택연금 금액은 **근사 계산**이며, 실제 수급액은
국민연금공단·한국주택금융공사의 공식 계산기 결과와 다를 수 있습니다.
감액률·가산률·물가상승률 등 제도 기준은 화면에서 조정 가능합니다.
